
# Neiramdal MISIS

Создание мотивационного модуля геймификации для кадровой системы «Алабуга»

## Frontend
Как запустить
```markdown
cd frontend
npm i
npm run dev
```

## Mobile
Как запустить
```markdown
cd mobile/neiramdal_misis
flutter pub get
flutter run
```

## Backend
Как запустить
```markdown
cd backend/NeiramdalMISIS
python3 manage.py runserver 
```
